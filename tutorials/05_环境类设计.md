# 教程五：环境类设计

## 5.1 环境类概述

环境类是 LeggedGym-Ex 的核心，负责：
- 与模拟器交互
- 计算观测值
- 计算奖励
- 检测终止条件

所有足式机器人环境都继承自 `LeggedRobot` 基类，而 `LeggedRobot` 又继承自 `BaseTask`。

## 5.2 LeggedRobot 基类详解

### 主要方法

让我们查看 `LeggedRobot` 类的核心方法：

#### `__init__` - 初始化

```python
def __init__(self, cfg: LeggedRobotCfg, sim_params: dict, sim_device, headless):
    self.cfg = cfg
    self.init_done = False
    self._parse_cfg(self.cfg)
    super().__init__(self.cfg, sim_params, sim_device, headless)

    self._init_buffers()
    self._prepare_reward_function()
    self.init_done = True
```

初始化步骤：
1. 解析配置
2. 调用父类初始化（创建模拟器）
3. 初始化缓冲区
4. 准备奖励函数

#### `step` - 环境步进

```python
def step(self, actions):
    actions = self._pre_sim_step(actions)
    self.simulator.step(actions)
    self.post_physics_step()

    # 裁剪观测值
    clip_obs = self.cfg.normalization.clip_observations
    self.obs_buf = torch.clip(self.obs_buf, -clip_obs, clip_obs)

    return self.obs_buf, self.privileged_obs_buf, self.rew_buf, self.reset_buf, self.extras
```

步进流程：
1. 预处理动作
2. 调用模拟器执行物理步进
3. 执行物理后处理（计算观测、奖励等）
4. 返回结果

#### `post_physics_step` - 物理后处理

```python
def post_physics_step(self):
    self.episode_length_buf += 1
    self.common_step_counter += 1

    self.simulator.post_physics_step()
    self._post_physics_step_callback()

    # 计算观测、奖励、重置...
    self.check_termination()
    self.compute_reward()
    env_ids = self.reset_buf.nonzero(as_tuple=False).flatten()
    self.reset_idx(env_ids)
    self.compute_observations()
```

这是最关键的方法，定义了每个时间步的处理流程。

### 常用可覆盖方法

`LeggedRobot` 提供了许多可以被子类覆盖的方法：

| 方法 | 说明 |
|-----|------|
| `_reset_dofs()` | 重置关节状态 |
| `_reset_root_states()` | 重置基座状态 |
| `compute_observations()` | 计算观测值 |
| `_get_noise_scale_vec()` | 获取观测噪声缩放 |
| `_post_physics_step_callback()` | 物理后处理回调 |

## 5.3 GO2 环境类示例

让我们分析 `GO2` 类的实现，了解如何自定义环境。

### 完整示例

```python
from legged_gym.envs.base.legged_robot import LeggedRobot
from .go2_config import GO2Cfg

class GO2(LeggedRobot):
    def _reset_dofs(self, env_ids):
        """重置关节位置和速度"""

        dof_pos = torch.zeros((len(env_ids), self.num_actions),
                              device=self.device, requires_grad=False)
        dof_vel = torch.zeros_like(dof_pos)

        # 髋关节添加小随机扰动
        dof_pos[:, [0, 3, 6, 9]] = self.simulator.default_dof_pos[:, [0, 3, 6, 9]] + \
            torch_rand_float(-0.2, 0.2, (len(env_ids), 4), self.device)

        # 大腿和小腿添加较大随机扰动
        dof_pos[:, [1, 4, 7, 10]] = self.simulator.default_dof_pos[:, [1, 4, 7, 10]] + \
            torch_rand_float(-0.4, 0.4, (len(env_ids), 4), self.device)
        dof_pos[:, [2, 5, 8, 11]] = self.simulator.default_dof_pos[:, [2, 5, 8, 11]] + \
            torch_rand_float(-0.4, 0.4, (len(env_ids), 4), self.device)

        self.simulator.reset_dofs(env_ids, dof_pos, dof_vel)

    def compute_observations(self):
        """计算观测值"""

        self.obs_buf = torch.cat((
            self.commands[:, :3] * self.commands_scale,        # 3: 命令
            self.simulator.projected_gravity,                   # 3: 重力投影
            self.simulator.base_ang_vel * self.obs_scales.ang_vel,  # 3: 角速度
            (self.simulator.dof_pos - self.simulator.default_dof_pos) * self.obs_scales.dof_pos,  # 12: 关节位置
            self.simulator.dof_vel * self.obs_scales.dof_vel,  # 12: 关节速度
            self.actions                                           # 12: 动作
        ), dim=-1)

        # 添加高度测量（如果需要）
        if self.cfg.terrain.measure_heights:
            heights = torch.clip(self.simulator.base_pos[:, 2].unsqueeze(
                1) - 0.5 - self.measured_heights, -1, 1.) * self.obs_scales.height_measurements
            self.obs_buf = torch.cat((self.obs_buf, heights), dim=-1)

        # 添加观测噪声
        if self.add_noise:
            self.obs_buf += (2 * torch.rand_like(self.obs_buf) - 1) * self.noise_scale_vec

    def _get_noise_scale_vec(self):
        """设置观测噪声的缩放向量"""

        noise_vec = torch.zeros_like(self.obs_buf[0])
        self.add_noise = self.cfg.noise.add_noise
        noise_scales = self.cfg.noise.noise_scales
        noise_level = self.cfg.noise.noise_level

        noise_vec[:3] = 0.  # 命令无噪声
        noise_vec[3:6] = noise_scales.gravity * noise_level
        noise_vec[6:9] = noise_scales.ang_vel * noise_level * self.obs_scales.ang_vel
        noise_vec[9:21] = noise_scales.dof_pos * noise_level * self.obs_scales.dof_pos
        noise_vec[21:33] = noise_scales.dof_vel * noise_level * self.obs_scales.dof_vel
        noise_vec[33:45] = 0.  # 动作无噪声

        return noise_vec

    def _reset_root_states(self, env_ids):
        """重置基座状态"""

        if self.simulator.custom_origins:
            base_pos = self.simulator.base_init_pos.reshape(1, -1).repeat(len(env_ids), 1)
            base_pos += self.simulator.env_origins[env_ids]
            base_pos[:, :2] += torch_rand_float(-0.5, 0.5, (len(env_ids), 2), self.device)
        else:
            base_pos = self.simulator.base_init_pos.reshape(1, -1).repeat(len(env_ids), 1)
            base_pos += self.simulator.env_origins[env_ids]

        base_quat = self.simulator.base_init_quat.reshape(1, -1).repeat(len(env_ids), 1)
        base_lin_vel = torch_rand_float(-0.0, 0.0, (len(env_ids), 3), self.device)
        base_ang_vel = torch_rand_float(-0.0, 0.0, (len(env_ids), 3), self.device)

        self.simulator.reset_root_states(env_ids, base_pos, base_quat,
                                          base_lin_vel, base_ang_vel)
```

### 观测向量组成

GO2 的观测向量（45 维）组成如下：

| 索引 | 内容 | 维度 |
|-----|------|------|
| 0-2 | 命令（线速度 x, y；角速度 yaw） | 3 |
| 3-5 | 重力投影向量 | 3 |
| 6-8 | 基座角速度 | 3 |
| 9-20 | 关节位置（相对于默认位置） | 12 |
| 21-32 | 关节速度 | 12 |
| 33-44 | 上一步动作 | 12 |

## 5.4 观测和特权观测

### 观测（Observation）

观测是策略网络的输入，只包含机器人在真实世界中能获取的信息：
- 关节状态
- IMU 数据（角速度、重力投影）
- 命令

### 特权观测（Privileged Observation）

特权观测包含额外的信息，用于：
- 教师网络（TS）
- 值函数估计
- 状态估计器训练

特权信息包括：
- 基座线速度
- 摩擦系数
- 附加质量
- 质心偏移
- 推力扰动

## 5.5 环境类设计最佳实践

### 1. 继承 LeggedRobot

始终从 `LeggedRobot` 继承，不要直接继承 `BaseTask`。

### 2. 只覆盖必要的方法

只覆盖需要自定义的方法，保持代码简洁。

### 3. 使用 PyTorch 张量操作

所有操作都应该使用 PyTorch 张量，并在正确的设备上：

```python
# 正确：使用 PyTorch 操作
dof_pos = torch.zeros((len(env_ids), self.num_actions), device=self.device)

# 错误：不要使用 NumPy
dof_pos = np.zeros((len(env_ids), self.num_actions))
```

### 4. 保持向量化

代码应该向量化，同时处理所有环境：

```python
# 正确：同时处理所有 env_ids
dof_pos[:, [0, 3, 6, 9]] = default_pos + random_noise

# 错误：不要循环处理每个环境
for i in env_ids:
    dof_pos[i, 0] = ...  # 不推荐！
```
