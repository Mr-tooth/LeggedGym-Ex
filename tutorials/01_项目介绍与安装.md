# 教程一：项目介绍与安装

## 1.1 项目概述

**LeggedGym-Ex** 是一个用于训练足式机器人的强化学习框架，基于 NVIDIA 的 legged_gym 开发。它支持多种模拟器和机器人平台，为足式机器人控制研究提供了完整的工具链。

### 主要特性

- **多模拟器支持**：IsaacGym、Genesis、IsaacSim (IsaacLab)
- **多种机器人**：Go2 四足机器人、G1 人形机器人、TRON1、双足步行器等
- **丰富的算法**：PPO、师生学习(TS)、显式估计器(EE)、DeepMimic 等
- **高性能**：支持大规模并行训练（最多 4096 个环境）
- **真实感模拟**：领域随机化、多种地形生成

### 支持的机器人和任务

| 机器人 | 任务类型 | 说明 |
|-------|---------|------|
| Go2 | go2 | 基础四足步行 |
| Go2 | go2_ts | 师生学习 |
| Go2 | go2_ee | 显式状态估计器 |
| Go2 | go2_dreamwaq | 隐式地形想象 |
| G1 | g1_mimic | DeepMimic 模仿学习 |
| TRON1 | tron1_pf | 工业四足机器人 |

## 1.2 环境要求

### Python 版本要求

- **IsaacGym**: Python 3.8
- **Genesis**: Python ≥ 3.10
- **IsaacLab**: Python ≥ 3.10

### 硬件要求

- NVIDIA GPU（推荐 RTX 3090 或更高）
- CUDA 11.8 或更高版本
- 至少 16GB 系统内存

## 1.3 安装步骤

### 步骤 1：克隆项目

```bash
git clone <repository_url>
cd LeggedGym-Ex
```

### 步骤 2：选择模拟器并安装

根据你的 Python 版本选择合适的模拟器：

#### 选项 A：使用 Genesis 模拟器（推荐，Python ≥ 3.10）

```bash
pip install -e .[genesis]
```

#### 选项 B：使用 IsaacGym 模拟器（Python 3.8）

```bash
pip install -e .[isaacgym]
```

#### 选项 C：使用 IsaacLab 模拟器

```bash
pip install -e .[isaaclab]
```

### 步骤 3：设置环境变量

```bash
# 设置模拟器（必需）
export SIMULATOR=genesis  # 或 isaacgym, isaaclab

# 可选：开启调试模式
export DEBUG=1
```

## 1.4 验证安装

运行以下命令验证安装是否成功：

```bash
# 查看帮助信息
python legged_gym/scripts/train.py --help
```

如果没有报错，说明安装成功！

## 1.5 项目目录结构

```
LeggedGym-Ex/
├── legged_gym/          # 主环境和模拟器接口
│   ├── envs/            # 环境实现
│   ├── simulator/       # 模拟器接口
│   ├── scripts/         # 训练和播放脚本
│   └── utils/           # 工具函数
├── rsl_rl/              # RL 算法和运行器
│   ├── algorithms/      # PPO 算法变体
│   ├── runners/         # 算法运行器
│   └── modules/         # 网络模块
├── resources/           # 机器人资产和参考运动
│   ├── robots/          # URDF 和网格文件
│   └── reference_motion/ # 参考运动数据
└── tutorials/           # 教程文档（本系列）
```
