# 教程二：快速开始训练

## 2.1 训练前准备

在开始训练之前，请确保你已经：

1. 完成了项目安装（见教程一）
2. 设置了正确的环境变量
3. 有可用的 NVIDIA GPU

### 检查环境变量

```bash
echo $SIMULATOR  # 应该输出 genesis, isaacgym 或 isaaclab
```

如果没有设置，请先设置：

```bash
export SIMULATOR=genesis  # 根据你的安装选择
```

## 2.2 运行第一个训练任务

### 基础训练命令

让我们从最简单的 Go2 四足机器人步行任务开始：

```bash
python legged_gym/scripts/train.py --task go2
```

这个命令会：

1. 创建 4096 个并行的 Go2 机器人环境
2. 使用标准 PPO 算法进行训练
3. 训练 1500 次迭代
4. 每 200 次迭代保存一次模型
5. 日志和模型保存在 `logs/go2/` 目录下

### 训练输出说明

训练开始后，你会看到类似这样的输出：

```
-------------------------------------------------------------------------------------------
|   iter    |  time / iter |  len / iter |   mean EPL   |   mean R   |   std R    |
-------------------------------------------------------------------------------------------
|    10     |    1.23s     |   122880    |    25.6      |    12.3    |    4.5     |
|    20     |    1.25s     |   122880    |    45.2      |    25.6    |    6.7     |
...
```

各列含义：
- `iter`: 迭代次数
- `time / iter`: 每次迭代耗时
- `len / iter`: 每次迭代的步数
- `mean EPL`: 平均回合长度
- `mean R`: 平均奖励
- `std R`: 奖励标准差

### 减少环境数量（调试用）

如果你的 GPU 显存不足，可以减少环境数量：

```bash
python legged_gym/scripts/train.py --task go2 --num_envs 1024
```

### 头less模式（无渲染）

在服务器上训练时，可以使用 headless 模式：

```bash
python legged_gym/scripts/train.py --task go2 --headless
```

## 2.3 其他常用训练任务

### 任务列表

项目包含多种预定义任务，以下是一些常用的：

#### 1. 师生学习 (Teacher-Student)

```bash
python legged_gym/scripts/train.py --task go2_ts
```

这个任务使用特权信息训练教师网络，然后指导学生网络。

#### 2. 显式估计器 (Explicit Estimator)

```bash
python legged_gym/scripts/train.py --task go2_ee
```

同时训练策略网络和状态估计器网络。

#### 3. G1 人形机器人模仿学习

```bash
python legged_gym/scripts/train.py --task g1_mimic
```

使用参考运动数据训练 G1 人形机器人。

#### 4. TRON1 工业机器人

```bash
python legged_gym/scripts/train.py --task tron1_pf
```

## 2.4 中断和恢复训练

### 中断训练

训练过程中可以随时按 `Ctrl+C` 中断训练，最新的模型会自动保存。

### 恢复训练

从之前保存的检查点恢复训练：

```bash
python legged_gym/scripts/train.py --task go2 --resume --load_run <run_name> --checkpoint <iteration>
```

例如：

```bash
python legged_gym/scripts/train.py --task go2 --resume --load_run Mar20_14-30-00_simple_rl_genesis --checkpoint 1000
```

## 2.5 可视化训练结果

### TensorBoard 可视化

训练过程中可以使用 TensorBoard 查看训练曲线：

```bash
tensorboard --logdir logs/go2
```

然后在浏览器中打开 `http://localhost:6006`。

### 播放训练好的模型

训练完成后，使用播放脚本查看效果：

```bash
python legged_gym/scripts/play.py --task go2 --load_run <run_name>
```

详细的播放和可视化方法将在**教程八**中介绍。

## 2.6 常见问题

### Q: 显存不足怎么办？

A: 减少环境数量或批量大小：

```bash
python legged_gym/scripts/train.py --task go2 --num_envs 1024
```

### Q: 训练速度太慢？

A:
1. 确保使用 GPU 训练
2. 使用 IsaacGym 或 Genesis 模拟器（IsaacSim 最慢但最真实）
3. 调整 `num_envs` 找到最佳性能平衡点

### Q: 奖励不上升？

A:
1. 检查奖励配置是否合理
2. 增加训练迭代次数（至少 500-1000 次）
3. 确认环境参数设置正确
